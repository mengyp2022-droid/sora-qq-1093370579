# 项目介绍

## 项目定位

`sora-qq-1093370579` 是一个面向本地部署场景的账号与任务管理项目，整体由协议脚本、Web 管理后台和本地数据存储三部分组成。项目将批量任务调度、资源池管理和后台操作界面整合到同一套代码中，便于在单机环境下集中管理。

## 整体结构

项目主要分为以下几个部分：

- `protocol_register.py`：处理协议注册相关主流程。
- `protocol_sora_phone.py`：处理 Sora 相关能力、手机号绑定及部分请求逻辑。
- `protocol_sentinel.py`：封装 Sentinel 相关逻辑。
- `main_protocol.py`：批量任务入口。
- `web/`：Web 管理端代码目录。
- `web/backend/app/`：FastAPI 后端接口与业务实现。
- `web/frontend/`：前端页面与静态资源。
- `docs/`：补充文档。

## 技术栈

- 后端：`FastAPI`
- 存储：`SQLite`
- 前端：原生 `HTML / CSS / JavaScript`
- 网络请求：`requests`、`curl_cffi`

## 使用方式概览

如果你主要是想先把项目跑起来，通常可以从以下入口开始：

```bash
pip install -r web/backend/requirements.txt
python web/run_web.py
```

默认情况下，Web 管理端会在本地启动，方便通过浏览器进入后台进行配置和查看任务状态。
